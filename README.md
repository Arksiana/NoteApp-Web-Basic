
# Basic Note WEB App with FASK

A brief description of what this project does and who it's for


## Installation and Setup

```bash
  gh repo clone Arksiana/NoteApp-Web-Basic
  pip install -r requirements.txt
```

  python main.py
  http://127.0.0.1:5000
